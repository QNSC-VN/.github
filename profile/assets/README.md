# Profile assets

Place the org-profile banner here.

- **File:** `banner.png` (or `banner.svg`)
- **Recommended size:** 1280 × 360 px (≈ 3.5:1). Keep important content centered;
  GitHub may scale it on different screens.
- **Content:** "QNSC — Quy Nhon Semiconductor" + the tagline
  *"Design Service Today. Vietnamese Chips Tomorrow."* on a branded background.
- Once added, it renders automatically at the top of `profile/README.md`.

> Tip: keep the file under ~1 MB so the profile page loads fast.
